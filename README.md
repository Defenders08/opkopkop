# ChubbyCMS

Лёгкий и понятный CMS движок на PHP + JS.

## Структура папок

```
/workspace/
├── index.php          # Главная страница сайта
├── style.css          # Стили
├── index.html         # Старая версия (можно удалить)
├── notes/             # Статьи хранятся здесь в виде JSON файлов
├── media/             # Медиафайлы (картинки, видео, аудио)
├── admin/             # Админ панель
│   ├── login.php      # Страница входа с 2FA
│   ├── index.php      # Главная админки
│   ├── editor.php     # Редактор статей
│   ├── editor.js      # JS для редактора
│   ├── media.php      # Управление медиа
│   ├── settings.php   # Настройки
│   ├── api_articles.php   # API для статей
│   ├── api_media.php      # API для медиа
│   └── api_settings.php   # API для настроек
└── includes/
    └── config.php     # Конфигурация и общие функции
```

## Установка

1. Скопируйте файлы на сервер с PHP
2. Убедитесь, что папки `notes` и `media` доступны для записи
3. Откройте сайт в браузере

## Вход в админку

- По умолчанию: пароль `admin123`, 2FA код `123456`
- Нажми **F2** на главной странице или перейди на `/admin/login.php`
- Вход двухэтапный: сначала пароль, потом 2FA код

## Функции

### Редактор статей
- Блочная структура (text, h3, img, img_little, ul, tab, hr, link, group)
- Вложенные блоки внутри img_little, tab, group
- Подборки и вкладки для организации статей
- Предпросмотр

### Медиа
- Загрузка drag-and-drop
- Поддержка: изображения, видео, аудио, PDF
- Все файлы хранятся в папке `media/`

### Безопасность
- 2FA аутентификация
- Защита от CSRF
- Санитизация входных данных
- Проверка типов загружаемых файлов
- Сессии с безопасными настройками

### Хранение данных
- Статьи: JSON файлы в папке `notes/`
- Медиа: файлы в папке `media/`
- Конфиг: `includes/config.json`

## Изменение пароля и 2FA

В разделе "Настройки" можно изменить:
- Пароль от админки
- 2FA код для входа

## Экспорт данных

В настройках есть кнопка экспорта всех данных в JSON.
